# pangu-test

This is a simple example to test [pangu.clj](https://github.com/coldnew/pangu.cl) in Clojure/ClojureScript.

## Usage (Clojure)

```
lein run -m pangu-test.core
```
## Usage (ClojureScript on Node.js)

```
lein cljsbuild once
node target/pangu-test.js
```
